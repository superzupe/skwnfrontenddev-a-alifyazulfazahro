# 🛋️ Dekoor — Furniture Landing Page

> A modern furniture landing page built with **React + TypeScript + Tailwind CSS + Framer Motion**.  
> Clean, aesthetic, and buttery-smooth animations — because UI should feel as good as it looks. 😌

---

## ✨ Features

- 🎨 Modern & aesthetic UI design
- 📱 Fully responsive (mobile & desktop)
- 🎞️ Smooth animations powered by Framer Motion
- 🛋️ Interactive product carousel (Best Sellers)
- 🧭 Category-based navigation
- 💌 Newsletter subscription section
- 🤝 Partnered brands showcase
- 🧱 Clean, modular component architecture

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| ⚛️ React | UI Framework |
| 🟦 TypeScript | Type Safety |

## 🚀 Getting Started

**1. Clone the repository**
```bash
git clone https://github.com/superzupe/skwnfrontenddev-a-alifyazulfazahro
```

**2. Navigate to the project directory**
```bash
cd dekoor
```

**3. Install dependencies**
```bash
npm install
```

**4. Start the development server**
```bash
npm run dev
```

**5. Open in browser**
```
http://localhost:5173
```
